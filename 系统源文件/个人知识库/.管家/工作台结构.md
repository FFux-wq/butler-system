# 工作台结构

> .管家/ 下每个文件是干什么的、大概什么格式。给管家参考，不是死模板。

## 文件说明

### 管家身份.md
管家启动时快速定位自己是谁。记录角色摘要——引领者+秘书，核心职责一句话。不写长篇，详细版见项目文档。

### 进度状态.md
当前做到哪了。记录当前项目、当前阶段、当前任务、上次更新时间。格式大概：

```
## 当前状态
- 当前项目：XXX
- 阶段：XX
- 当前任务：XXX
- 上次更新时间：YYYY-MM-DD

## 项目进度
- [x] 已完成
- [ ] 下一步
```

上次更新时间用于启动时的时效性校验——7天内直接进，7~30天提醒同步，超30天建议体检。

### 任务队列.md
所有任务的列表，分三类：

```
## 进行中
- 项目名（一句话描述）

## 待办
- 待办事项

## 已完成
- 已完成事项（日期）
```

### 时间线.md
按日期追加关键事件——项目推进、系统改造、重要决策。每次对话结束追加本次记录。

```
## YYYY-MM-DD

### 项目进展
- 项目名：做了什么

### 系统改造
- 改了什么
```

### 用户偏好.md
管家观察到的用户偏好。管家自己写的，不覆盖用户自己写的画像。

```
- 偏好内容
- 常见卡点
- 工作习惯
```

### 可用工具卡.md
触发词 → 工具 的速查表。管家判断用户意图后快速匹配。

### 管家须知手册.md
管家的行为准则——工作节奏、交互边界、教不是替。

### 工作台结构.md
就是本文件。管家自己看自己。
