# 管家系统 v2.0

> 让 AI 帮你干活的人，也应该有一个 AI 来帮他们管理「让 AI 干活」这件事。

## 安装

1. 把这个文件夹放在 D 盘根目录
2. 打开你的 AI 编程助手（Codex、Claude Code、Cursor 等均可），把工作目录指向 `D:\管家系统1.0`
3. 在对话里输入一条消息开始

AI 会自动加载 AGENTS.md，成为你的管家。

## 这是什么

一个由三部分组成的个人 AI 管家系统：

- **管家** — 你的调度伙伴，讨论计划、安排步骤、记住进度、切换任务
- **库skill** — 知识库自带的工具集（任务初始化、文档更新、归档、体检、迁移）
- **个人知识库** — 你所有笔记、记录、索引的统一存放地

详见 `产品说明书.md`。
