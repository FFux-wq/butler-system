# 管家系统 + 个人知识库

## 项目介绍

### 为什么要做这个项目

因为你学的东西越来越杂，做的事越来越多。

你做完智能锁和云家居两个项目后，知识笔记散落在四个目录里——个人知识库、项目缓存、MCU 资料库、复习文件夹。找一个东西要翻好几个地方，时间久了连自己写过什么都忘了。

同时你手上的事开始并行——要复习面试、要整理项目、要学新东西、可能还要接新项目。靠自己脑子记，迟早会漏。

你需要一个东西帮你看住这一切。

### 这个项目能帮你做什么

**它把两件事合在一起：**

**个人知识库** — 你所有笔记、记录、源代码和项目文档的统一存放地。代码和文档在一起，不用额外维护外部路径索引。由库skill 负责维护它的秩序。

**管家** — 跟你一起工作的搭档。它记住你在做什么、做到哪了、准备做什么。每次打开对话，它自动进入状态，告诉你当前进度和下一步建议。

### 两者怎么配合

```
你起了一个新任务
  → 跟管家讨论计划
  → 管家记下来，引导你执行
  → 你干活时，管家从知识库调资料给你
  → 干完了，管家更新进度
  → 需要整理时，管家调用库skill 维护知识库
  → 下次打开，管家直接接上
```

知识库是管家的"资料库"和"代码库"，管家通过固定的路径指针访问它。两者独立但协同工作。

### 这个项目不做的事

- 不做成通用产品——它是为你一个人设计的
- 不需要服务器、数据库、部署——就是一堆 Markdown 文件 + AI 对话
- 不追求完美才开始——边用边改
